
# Windows电脑使用clash后,不开clash没网

有一部分系统关机时不先关闭Clash，可能会导致上述问题。
解决方法：
- 请重新打开clash后再手动关闭一次。
- 打开电脑的设置，手动关闭电脑的代理服务器开关。
